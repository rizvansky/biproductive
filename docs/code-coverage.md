How to test our program (reproduce the results):

1. Run `$ docker-compose up --build`
2. After the application is launched run in separate terminal: `$ docker exec -it <name-container> bash`

- name of container could be found by command `$ docker ps` (in my case. name of container is biproductive_web_1).

3. Inside of container

- run `pip install coverage`
- run `cd biproductive; coverage run --source='.' manage.py test`
- run `coverage report`
